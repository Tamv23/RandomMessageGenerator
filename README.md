# RandomMessageGenerator

A small JavaScript project that generates **motivational and positive messages**. Every time you run the program, it outputs a new random set of messages for Mindset, Physical activity, and Spiritual encouragement.  

This project was built as part of a Codecademy Full-Stack Engineer course assignment.

---

## Features

- Generates random motivational quotes in three categories: Mindset, Physical, and Spiritual.
- Uses JavaScript objects and arrays to store quotes.
- Includes a helper function to select random quotes.
- Logs the output to the console for easy testing.
- Demonstrates use of Git version control and branching workflow.
